# Auxidien (AUXI)

Auxidien is an on-chain, volatility-weighted DeFi index designed to reflect the dynamics of precious metals markets.

This repository contains documentation, methodology notes, and supporting materials related to the Auxidien project.

## Overview
- Index-based DeFi design
- Volatility-aware methodology
- Transparency-first approach
- No price targets, no guarantees
